what it is

KONTUR (kohn-toor) is a terminal-native workstation where two engineers jointly supervise a fleet of coding agents on the same codebase. One hosts the agents, one joins to co-sign; both review as co-equal checkers. Every change an agent makes is gated behind two independent human approvals before it can merge — and the whole session leaves a signed, tamper-evident audit trail.

It exists for the place solo agentic tooling can't go: production environments where the agent wrote it and one person glanced at it isn't good enough, and segregation of duties is the rule, not a nicety.

how it works — four gates

01
dispatch
both seats compose the prompt in-console; every edit resets consent. Both mark ready to dispatch.
02
plan
the agent proposes a task list of bounded, single-concern changes and blocks until both approve.
03
gate
each finished change parks at a dual-hold. First key sealed, second cast blind. A no-go carries its fix.
04
merge
the approved set lands as one reviewed commit with Reviewed-by trailers and a hash-chained audit head.

install

Distributed as a Homebrew tap (macOS + Linux):

$ brew tap industrial-assets/kontur
$ brew trust industrial-assets/kontur   # once: recent Homebrew requires trusting third-party taps
$ brew install kontur

Upgrade when a new version ships — brew upgrade kontur. Or build from source:

$ git clone https://github.com/industrial-assets/kontur
$ cd kontur
$ cargo install --path crates/kontur-tui --bin kontur

Then cd your-project && kontur to host, and kontur join kontur://… to co-sign. A fresh folder works too — Kontur offers to git init and makes a base commit.

design principles

  • Every element is decision-relevantIf an operator doesn't read it to decide or act, it isn't on screen. No telemetry, no confidence scores, no ornament.
  • Emphasis is spent onceThe single thing that needs a human is loud; everything else is calm. A screen where three things shout is a screen where nothing does.
  • Calm is the defaultThe authentic control-room principle is reducing alarm noise, not manufacturing it. Kontur is quiet until it needs you.
  • Keyboard-firstEvery action has a key, shown inline. The command line at the bottom is always live. No hidden verbs.
  • The look is a consequence, not a costumeDensity and monospace fall out of supervising many things at once. The Cyrillic/red flourish stays in the identity register.
  • Two operators, one truthBoth see the same authoritative fleet state — except where independence requires divergence: the blind sign-off.
